备份和还原

备份/还原

备份

你可以创建卡片日记的备份文件来对卡片日记进行手动备份，创建了备份文件以后请保存卡片日记的备份文件到其他平台（防止卸载卡片日记导致丢失备份文件）。

点击备份的 zip 文件可把文件分享到其他平台（如 iCloud 文件）。

还原

你可以从卡片日记的备份文件中还原你的日记数据。 你也可以把你在 Day One 或者 Journey 写的日记数据恢复到卡片日记中。

从 iCloud Drive，Dropbox 等云端还原

点击 导入 - iCloud, Dropbox 等 选择对应的 ZIP 文件

从 iTunes 还原

要从 iTunes （macOS 系统现已集成到 Finder 中）还原，请连接你的手机到装有 iTunes 或者（macOS 系统）的电脑 1. 点击你的设备 2. 选择 文件 3. 展开 卡片日记 4. 把要还原的 ZIP 文件拖入到卡片日记的根目录下（请不要移动其他任何文件） 5. 打开卡片日记 App - 设置 - 备份 / 还原 - 导入 - iTunes - 导入

Característica da etiqueta

Adicionar ou editar etiquetas

  • Na vista de edição, clique na imagem do TAG abaixo da coluna "Titulo".
  • Selecione TAGs na página pop-up
  • Clique no botão + para criar um novo separador
  • Seleccione as etiquetas que pretende (Pode escolher várias etiquetas)

Qual é o propósito das Tags?

Na página "Minha", pode ver as suas agendas por separadores. Pode contar o número de diferentes tipos de agendas com base na etiqueta.

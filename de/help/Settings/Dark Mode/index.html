Dunkler Modus

Card Diary unterstützt einen sehr ästhetisch ansprechenden Dark-Modus. Alle Seiten von Card Diary unterstützen den Dark-Modus.

  • Schalten Sie den Dark-Modus ein, indem Sie auf den Schalter
  • Fügen Sie eine Zeitspanne hinzu, um den Dunkelmodus automatisch zu aktivieren, oder Sie können dies auf der Grundlage des Systems tun.
